# Quiz Game CLI 

A fun **command-line** football (soccer) quiz application built in Python.

Test your knowledge with questions about players, teams, matches, history, and more!

## Features
- Multiple-choice questions
- Score tracking
- (Add more as you have: timed rounds? Categories? Save progress?)

## Project Structure
app/                # Main application logic
├── models/         # Data models (e.g. User, Question)
├── services/       # Quiz/business logic
├── core/           # Core utilities
├── main.py         # Primary entry point
└── run.py          # Launcher script (if separate)data/               # Data handling
└── database.py     # Loads questions, handles storage


## How to Run
1. Make sure you have **Python 3.8+** installed
2. Clone the repo:
   ```bash
   git clone https://github.com/Variant1740/Quiz_game_CLI.git
   cd Quiz_game_CLI
3.Run the app:
  python app/run.py
  or
  python app/main.py
  
## Requirements

- Python 3.8 or higher
- No external packages or libraries required
- Uses only built-in Python modules (e.g. `random`, `json`, etc.)
